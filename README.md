## ‚öôÔ∏è Setup & Installation

```text
1. Download the CareerCompass project from the repository.
2. Extract the ZIP file to any folder on your local system.

3. Open Command Prompt or Terminal.
4. Navigate to the backend directory:
   cd backend

5. Ensure Python is installed and added to the system PATH.

6. Install all required Python dependencies:
   pip install -r requirements.txt

7. Open the following link in your browser to generate a Gemini API key:
   https://aistudio.google.com/app/u/1/api-keys

8. Create a new Gemini API key and copy it.

9. Inside the backend folder, create a file named .env

10. Add the following line inside the .env file:
    GEMINI_API_KEY=your_api_key_here

11. Save the file.

12. From the backend folder, run the Flask server:
    python app.py

13. Verify the backend is running at:
    http://127.0.0.1:5000

14. Navigate to the frontend folder.

15. Open any HTML file (for example: home.html).

16. Copy the full file path and paste it into a web browser.

17. Use the application to:
    - Paste your resume
    - Analyze job descriptions
    - Get AI-powered job suggestions
